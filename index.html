GetFit: Nutrition Website
Python Flask JavaScript jQuery API

GetFit is a nutrition website that allows users to calculate their daily nutrient needs, build healthy meal plans, and lookup nutrition facts for their favorite foods. The site is built using Python and Flask on the backend, Javascript and jQuery of the front, and styled with Bootstrap. Nutrition information is pulled from the Nutritionix API.

HarvardX

This course teaches you to teach yourself programming. The course starts by holding your hand for 5 weeks, introducing data structures, memory allocation, and algorithms through projects coded in C. In the following 6 weeks you are completing projects using Python, Flask, HTML, CSS, JavaScript, jQuery, and SQL. This course is an excellent introduction to computer science and programming fundamentals and provides a foundation to build upon.
