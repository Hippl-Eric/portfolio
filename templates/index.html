Review-a-Book: Website
Python Flask postgreSQL API

Review-a-Book is a Flask application which allows users to search for ratings and reviews for up to 5,000 books. The site accesses a postgreSQL database hosted on Heroku. In addition to user reviews, additional rating information is provided by the Goodreads API. Lastly, the site is accessible via its own API route, and will return book information in JSON format with a valid request.
